# ShoPperZ

An e-commerce frontend website built with HTML, CSS, and JavaScript.

## Features
- Responsive navigation with hamburger menu for mobile
- Auto-sliding image banner
- Product grid with 18+ items across categories
- Hover zoom effects on product images
- Footer with social media links and site sections

## Tech Stack
- HTML5
- CSS3 (animations, flexbox, responsive media queries)
- JavaScript (DOM manipulation)
